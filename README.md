# 💼 Ajayghosh P – Visual Resume

This is a fully responsive **visual resume** built using **HTML**, **TailwindCSS**, and **Chart.js** to showcase my professional profile, career journey, and FinTech aspirations.

### 🔍 Features

- 📊 **Skills Radar Chart** – Highlights core competencies in finance, analytics, compliance, and tech.
- 🧠 **Strategic Path to FinTech** – Visual roadmap from CMA qualification to Financial Analyst goal.
- 🏢 **Work Experience & Impact** – KPI-driven role summaries with real transaction metrics.
- 📜 **Certifications & Education** – Graphs and icon-based summaries of academic milestones.
- 🌐 **Hosted via GitHub Pages**

### 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **HTML5** | Base structure |
| **TailwindCSS** | Styling and responsive design |
| **Chart.js** | Dynamic data visualizations |
| **GitHub Pages** | Free hosting platform |
